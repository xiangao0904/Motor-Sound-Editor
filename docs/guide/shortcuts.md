# Shortcuts

Motor Sound Editor already supports a small set of high-frequency shortcuts. Once you learn them, the daily editing loop becomes much faster.

## Global editing shortcuts

- `Space`: play / pause preview
- `Ctrl+S`: save
- `Ctrl+Shift+S`: save as
- `Ctrl+Z`: undo
- `Ctrl+Y`: redo
- `Ctrl+Shift+Z`: redo
- `Ctrl+D`: clear the current selection
- `Delete`: delete the selected keyframe
