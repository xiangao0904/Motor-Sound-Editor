# Inspiration and Credits

Graphical keyframe editing of motor sounds is not a new concept. Before the development of this project, several mature tools in the simulated railway community had already validated the feasibility of this logic. The Motor Sound Editor's design philosophy references classic works from BVE workshop (Motornoise Audition), zbx1425 (keyframe editor), and hi03 (RTMSoundMaker).

* **Motornoise Audition** ([BVE workshop](https://bvews.jpn.org/materials/motornoise-audition/)): Provides a basic logical reference for motor sound synthesis and preview.

* **Keyframe Motor Sound Editor** ([zbx1425](https://www.zbx1425.cn/nautilus/index.html)): Its efficient keyframe editing concept greatly inspired the workflow design of this software.

* **RTMSoundMaker** ([hi03](https://x.com/hi03_s/status/1700218889437851852)): Although originating from the RTM community, its support for the BVE format provided valuable insights for this project.

As part of the community's technological evolution, this project aims to reimplement and extend this workflow using a modern technology stack (Tauri + Vue 3), providing superior performance and interactive experience while remaining compatible with the BVE specification. We express our respect to the pioneers mentioned above.
