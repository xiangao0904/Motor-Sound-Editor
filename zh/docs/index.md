# 用户文档

Motor Sound Editor 是一款专注于电机音效工程的桌面编辑器，支持可视化曲线编辑、按轨道分层分配音频、实时试听与导出打包。本文档面向希望稳定完成创建、编辑、校对和导出的终端用户。
