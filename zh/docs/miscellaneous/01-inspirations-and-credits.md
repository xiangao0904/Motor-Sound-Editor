# 灵感与鸣谢

电机音的图形化关键帧编辑并非新概念。在开发本项目前，模拟铁路社区已有数款成熟工具验证了这一逻辑的可行性。Motor Sound Editor 在设计理念上参考了 BVE workshop (Motornoise Audition)、zbx1425 (关键帧式编辑器) 以及 hi03 (RTMSoundMaker) 的经典工作。

* **Motornoise Audition** ([BVE workshop](https://bvews.jpn.org/materials/motornoise-audition/)): 提供了电机音合成与预览的基础逻辑参考。
* **关键帧式电机音编辑器** ([zbx1425](https://www.zbx1425.cn/nautilus/index.html)): 其高效的关键帧编辑理念极大地启发了本软件的操作流设计。
* **RTMSoundMaker** ([hi03](https://x.com/hi03_s/status/1700218889437851852)): 尽管源自 RTM 社区，但其对 BVE 格式的支持为本项目提供了宝贵的思路。

作为社区技术演进的一环，本项目旨在通过现代化的技术栈（Tauri + Vue 3）重新实现并扩展这一工作流，在兼容 BVE 规范的同时，提供更极致的性能与交互体验。在此向上述先行者的探索表示敬意。
