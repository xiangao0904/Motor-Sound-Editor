# 项目详情与保存

本页说明项目详情弹窗中可以查看的信息、`.msep` 项目格式的作用，以及保存与未保存变更的当前行为。

## 1. 项目详情

![项目信息](/docs/ProjectDetails.png)

**Project Details** 会展示当前工程的项目名称和一组统计信息。当前实现中：

- **可编辑项**：项目名称。
- **只读信息**：文件路径、轨道数、音频资源数、已分配音频数、关键帧总数、最高速度、加速度、制动减速度、创建时间和更新时间。

这些信息有助于你在继续编辑或准备导出前确认当前工程状态。

## 2. `.msep` 文件格式

Motor Sound Editor 使用 `.msep` 作为专属工程文件格式。

- **一体化打包**：`.msep` 会把项目元数据、轨道定义、曲线数据和已打包的音频资源一起保存。
- **便于迁移**：分享或移动 `.msep` 时，不需要额外整理一组松散的音频文件。
- **持续编辑**：`.msep` 是可编辑源文件，而不是最终交付包。

## 3. 保存与未保存提示

- **未保存状态**：当你添加轨道、修改关键帧、替换音频或更新项目名称时，项目会进入未保存状态。
- **保存（Save）**：把当前文档与音频打包写回原来的 `.msep` 文件。
- **另存为（Save As）**：将当前工程写入新的 `.msep` 文件副本。
- **离开前提示**：如果项目存在未保存改动，返回首页或关闭窗口前会弹出保存确认。
- **撤销与重做**：保存不会清空当前会话内的撤销 / 重做历史；只要应用未关闭，你仍可继续回退近期操作。
