# 界面总览

Motor Sound Editor 将项目管理与工程编辑明确拆分成两个主要空间，以保持工作流清晰：先在首页管理项目，再进入编辑器完成声音创作。

## 首页（Home）：项目生命周期管理

![标注首页](/docs/labledhome.png)

首页是项目流程的起点，主要负责：

- **工程索引**：快速查找和回到已有工程。
- **资产调度**：执行新建、导入和打开工程文件等操作。
- **最近项目管理**：通过搜索、排序和卡片预览整理工作集。

## 编辑器工作区（Editor Workspace）：参数编辑与试听

![标注编辑页面](/docs/lablededit.png)

编辑器工作区是软件的核心，重点在于：

- **层级控制**：明确当前激活的轨道层与音频资源。
- **实时预览**：在不同速度和状态下即时试听音高（Pitch）与音量（Volume）的变化。
- **数据修改**：对关键帧进行可视化增删、移动与数值微调。
