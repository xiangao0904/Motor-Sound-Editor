# 导出流程

当你完成音轨编辑和曲线调整后，就可以将当前工程导出为目标模拟器可用的打包产物。

## 1. 导出前检查项

在进行导出之前，请确认：

- **有效音轨**：导出引擎只会处理 **已启用（Enabled）**、**未静音（Unmuted）** 且 **已分配音频资源** 的轨道。
- **至少存在一条可导出轨道**：如果所有可导出轨道都没有分配音频资源，导出会直接报错。

## 2. 导出对话框

![项目导出](/docs/export.png)

点击导出按钮后，会弹出 **Export File** 对话框，你可以在这里配置最终输出的格式和音频参数。

## 3. 导出格式

- **BVE**：当前唯一可用的导出格式。选择后，软件会自动生成标准的 **ZIP 压缩包**，例如 `[项目名称]-BVE.zip`。
- **OpenBVE / MTR**：目前仍是禁用状态，仅作为未来扩展目标的占位项显示在界面中。

## 4. 采样率

你可以为导出的音频指定全局采样率。

- 可选值包括：`22050 Hz`、`32000 Hz`、`44100 Hz`、`48000 Hz`、`96000 Hz`。
- 默认推荐值通常是 `44100 Hz`，能在音质与文件体积之间取得较好的平衡。

## 5. 格式转换与自动适配

Motor Sound Editor 原生支持导入 `.ogg` 和 `.wav` 音频。为了保证导出结果符合目标格式要求，软件会在导出时自动执行必要的转换。

如果项目中存在 `.ogg` 音频，而导出目标需要 WAV，导出对话框底部会显示提示信息，说明这些 OGG 文件会在导出时自动转换成 WAV。

你无需手动转码。确认配置无误后，点击 **Export** 即可完成导出。
