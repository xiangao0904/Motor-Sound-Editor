# 列表编辑器

图表编辑器是主要创作界面，但不是唯一入口。Motor Sound Editor 同时提供 **列表编辑器（List Editor）**，用于以表格方式处理关键帧数据。

当你需要批量查看、校对或精确输入大量关键帧时，列表编辑器会比图形化曲线提供更直接的数据反馈。

![列表视图](/docs/listview.png)

## 1. 核心功能

列表编辑器会把当前轨道的关键帧映射成表格内容。通过这种方式，你可以像处理电子表格一样管理声音数据。

- **精确输入**：可以直接输入具体数值，例如 `60.5 km/h` 或 `0.85`。
- **全局总览**：在一个视图中查看当前轨道的关键帧分布，更容易发现突跳和冗余点。
- **双工况并排编辑**：当前实现会同时显示 **Traction** 和 **Brake** 两组数据，并分别列出 Pitch 与 Volume。

## 2. 界面组成

表格主要由以下区域构成：

1. **Traction 区域**：包含牵引模式下的速度与数值列。
2. **Brake 区域**：包含制动模式下的速度与数值列。
3. **Pitch / Volume 列**：分别对应音高与音量关键帧。

## 3. 关键特性

### 草稿式编辑

列表编辑器采用草稿式编辑流程：

- 在列表中修改内容时，变更先停留在弹窗内部。
- 点击 **Apply** 后，变更才会正式写回项目。
- 点击 **Cancel** 则会放弃当前草稿。

### 自动排序

关键帧会始终按速度从小到大排序。当你修改某个关键帧的速度后，列表中的行序可能自动重排，以保持逻辑顺序一致。

## 4. 右键上下文菜单

无论是在图表工作区还是列表编辑器中，你都可以通过右键呼出上下文菜单，快速执行常用操作。

![右键上下文菜单](/docs/listcontexmenu.png)

当前列表编辑器中的常见右键操作包括：

- **同步 traction 到 brake / 同步 brake 到 traction**：将一侧数据复制到另一侧。
- **添加关键帧**：快速插入新的关键帧，并自动参与排序。
- **删除当前关键帧**：移除当前选中的关键帧。
